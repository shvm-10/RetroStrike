# ⚡ RetroStrike

A retro-style arcade survival game built with vanilla HTML, CSS and JavaScript.

Survive waves of enemies, collect power-ups, defeat bosses and beat your high score.

🎮 **[Play Now](https://shvm-10.github.io/RetroStrike)**

## 🕹️ How to Play

| Key | Action |
|-----|--------|
| W / A / S / D | Move |
| ENTER | Auto-aim shoot |
| R | Restart |
| F | Fullscreen |
| ESC | Exit Fullscreen |

---

## ✨ Features

- 👾 Endless enemy waves that get faster every level
- 💀 Boss enemy spawns every 3 levels
- ⚡ Power-ups — Speed Boost, Triple Shot, Shield
- 🏆 Local high score leaderboard (top 5)
- 🌙 Dynamic background cycle
- 💥 Particle explosions and bullet trails
- 🔊 Retro sound effects (Web Audio API)
- 📺 CRT scanline effect
- 🕹️ Arcade cabinet UI

---

## 🛠️ Built With

- HTML5 Canvas
- CSS3 Animations
- Vanilla JavaScript
- Web Audio API
- Google Fonts (Press Start 2P)
